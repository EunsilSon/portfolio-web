# PORTFOLIO-WEB

# 프로젝트 소개
프론트엔드 공부를 위해 개발한 첫 프론트엔드 프로젝트이며, **포트폴리오** 목적의 원페이지 웹사이트입니다.  

AWS EC2에서 Apache2 서버를 구축해 배포했으며, AWS Route 53을 이용해 도메인을 적용했습니다.

<br>

# 사용 기술
- AWS EC2
- AWS Route 53
- Apache2
- HTML/CSS
- Vanila JS
