북마키 : 독서 기록 도우미

2024.07 - 2024.11 (1人 개인 프로젝트)

원하는 책을 검색하고 마음에 들었던 구절을 기록하는 독서 서비스입니다. 네이버 Open API를 통해 책 검색이 가능하고, 구절 작성 시 책 정보가 자동 입력됩니다. 책별로 구절을 저장할 수 있습니다.
Spring Security 폼 로그인을 통해 인증을 구현하고, SecurityContext의 인증된 Authentication에서 username을 추출해 비즈니스 로직에 사용했습니다.
안전한 비밀번호 변경을 위해 이메일로 변경 링크를 전송하고 Redis 토큰을 이용해 링크에 접근하는 대상의 신원 확인을 수행했습니다. 토큰 탈취를 방지해 보안 질문을 통해 2차 검증을 수행했습니다.
Validation을 적용해 사용자 데이터의 제약 조건 검증과 예외 처리를 구현했습니다.
Soft Delete를 적용해 최근 삭제한 구절 조회 및 복구 기능을 구현했습니다. Hibernate 동적 필터링으로 조회하고, @Scheduled로 삭제한지 30일 지난 데이터를 자동으로 영구 삭제했습니다.
로깅 작업에 AOP를 적용해 호출된 API와 메서드 실행 시간을 로깅하여 병목 현상을 파악했습니다.
Spring 내부 캐시를 활용해 책 개수 체크 기능을 구현했습니다. 변동 가능성은 낮지만 자주 조회되는 값을 캐싱해, DB 부담을 최소화하고 성능을 약 98% 향상시켰습니다. (174ms → 3ms)

책 검색, 구절 작성 · 수정 · 삭제 · 조회, 구절 실시간 개수 조회

▸ Front-end

TypeScript, Axios

▸ Back-end

Java 17, Spring Boot 3, Spring Security, Spring Data JPA

▸ Deployment

AWS EC2, AWS RDS, Docker, MySQL, Redis

One-Pass
